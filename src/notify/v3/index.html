Documentação - Biblioteca Notify V3

Este documento explica como utilizar a biblioteca de notificações para exibir mensagens no navegador de forma simples e eficaz.

Instalação

Adicione o seguinte script no <head> ou no final do <body> da sua página:

<script defer src="https://libsme.vercel.app/src/notify/v3/index.js" type="module"></script>

Exemplo Mínimo

Copie o código abaixo no <body> do seu arquivo HTML:

<button onclick="mostrarNotificacao('Notificação Default')">Default</button>
<script defer src="https://libsme.vercel.app/src/notify/v3/index.js" type="module"></script>

Como Usar

Use a função mostrarNotificacao(mensagem, opcoes?) para exibir notificações. Exemplo:

<script>
    mostrarNotificacao("Esta é uma notificação padrão!");
</script>

Com Opções Personalizadas

<script>
    mostrarNotificacao("Notificação Personalizada!", {
        cor: "#4CAF50",
        duracao: 4000,
        movimentoEntrada: "deslizar",
        movimentoSaida: "esvair",
        posicao: "bottom-left"
    });
</script>

Personalize suas notificações usando as propriedades abaixo:

Configurações Disponíveis

Propriedade Tipo Valor Padrão Descrição
cor string #333333 Define a cor de fundo da notificação.
duracao number 5000 Tempo em milissegundos antes da notificação desaparecer.
movimentoEntrada string "deslizar" Animação de entrada da notificação.
movimentoSaida string "esvair" Animação de saída da notificação.
posicao string "top-right" Define a posição da notificação na tela.
